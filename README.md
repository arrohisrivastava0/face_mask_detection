# Face Mask Detection using OpenCV

## Overview
This project detects faces in real-time using a webcam and classifies whether a person is wearing a mask or not.

## Features
- Real-time face detection
- Mask / No Mask classification
- Runs via command line

## Requirements
- Python 3.x
- OpenCV
- NumPy

## Installation

pip install -r requirements.txt

## Run the project

python main.py

## Project Structure
- main.py : main script
- haarcascade_frontalface_default.xml : face detection model
